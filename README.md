# 🌐 My Portfolio

🌐 **Live Demo:** https://your-vercel-link.vercel.app

💼 **Personal Developer Portfolio** | Modern UI & Smooth Animations
📂 **GitHub Repository:** https://github.com/BhaveshWasnik/MyPortfolio

My Portfolio is my personal developer portfolio built using **React.js**, **Vite**, **Tailwind CSS**, and **Framer Motion**. It showcases my skills, projects, technical expertise, and contact information through a clean, responsive, and interactive user interface.

This project was created to establish a professional online presence, highlight my development journey, and demonstrate my frontend development skills with modern web technologies.

---

# 🚀 Features

✨ Modern and professional portfolio design

📱 Fully responsive across desktop, tablet, and mobile devices

🎬 Smooth animations powered by Framer Motion

🎨 Beautiful UI built with Tailwind CSS

💼 Dedicated Projects Showcase

🧑‍💻 Skills & Technologies Section

📧 Contact section with quick email access

⚡ Fast performance using Vite

🌙 Interactive visual effects and modern design

# ⚙️ Setup and Installation

## 🔹 Step 1: Clone the Repository

```bash
git clone https://github.com/BhaveshWasnik/MyPortfolio.git
```

## 📁 Step 2: Navigate to the Project Folder

```bash
cd MyPortfolio
```

## 📦 Step 3: Install Dependencies

```bash
npm install
```

## ▶️ Step 4: Run the Development Server

```bash
npm run dev
```

## 🌐 Step 5: Open in Browser

Open your browser and visit:

```text
http://localhost:5173
```
